# WAP_test

Requierments:
- Python 3.11.10
- ChromeDriver 130.0.6723.58
- Google Chrome 130.0.6723.58 

Install required packages:
```
run pip install requests selenium pytest
```

Run tests locally in CMD: 
```
pytest src/main.py
```
